# CircleHub

CircleHub is a front-end social media prototype inspired by modern community platforms.

## Features

- Personalized profile panel with connection tracking
- Create posts with mood tags
- Like and comment interactions
- Search and filter feed posts in real time
- Suggested connections with one-click connect
- Quick messaging panel with local history
- Communities, events, and trending tags sections
- Light/dark theme toggle
- Persistent app state using `localStorage`

## Run locally

Because this is a static web app, serve the directory with any local server:

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.
